# 🛒 МАГАЗИН — Онлайн магазин на PHP + MySQL

Полноценный онлайн-магазин в стиле DNS/Ситилинк.
Стек: PHP (без фреймворков) + MySQL + HTML/CSS/JS.

---

## ⚡ Быстрый старт (XAMPP)

### 1. Установка XAMPP
Скачайте и установите [XAMPP](https://www.apachefriends.org/).
Запустите **Apache** и **MySQL** из панели XAMPP Control Panel.

### 2. Размещение проекта
Скопируйте папку `shop` в:
```
C:\xampp\htdocs\shop\
```

### 3. Создание базы данных
1. Откройте браузер: http://localhost/phpmyadmin
2. Нажмите **"Новая"** (New) в левой панели
3. Назовите базу: `shop_db`, кодировка `utf8mb4_unicode_ci`
4. Нажмите **Создать**
5. Выберите `shop_db` → вкладка **"Импорт"**
6. Выберите файл `database.sql` из папки проекта
7. Нажмите **"Вперёд"** (Go)

### 4. Настройка подключения к БД
Откройте файл `includes/db.php` и при необходимости измените:
```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'shop_db');
define('DB_USER', 'root');
define('DB_PASS', '');        // По умолчанию в XAMPP пароль пустой
```

### 5. Создание папки для загрузок
Убедитесь что папка `uploads/` существует и доступна для записи.
Если нет — создайте вручную:
```
shop/
  uploads/
    avatars/
    products/
    news/
    promos/
```

### 6. Запуск
Откройте в браузере:
```
http://localhost/shop/
```

## ⚙️ Часто задаваемые вопросы

**Q: Изображения не загружаются**
A: Проверьте что папка `uploads/` и её подпапки существуют и доступны для записи (chmod 755 или 777).

**Q: Ошибка подключения к БД**
A: Убедитесь что MySQL запущен в XAMPP и база `shop_db` создана. Проверьте настройки в `includes/db.php`.

**Q: Не отправляются письма**
A: В XAMPP mail() не работает без настройки. Для локальной разработки это нормально — письма можно логировать или использовать [MailHog](https://github.com/mailhog/MailHog).

**Q: Как сменить пароль администратора?**
A: Войдите как admin, перейдите в личный кабинет — или обновите напрямую в phpMyAdmin:
```sql
UPDATE users SET password = '$2y$10$ВАШ_ХЕШ' WHERE login = 'admin';
```
Хеш генерируется через PHP: `echo password_hash('новый_пароль', PASSWORD_DEFAULT);`
